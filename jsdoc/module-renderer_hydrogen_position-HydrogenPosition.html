Class: HydrogenPosition

renderer/hydrogen_position~HydrogenPosition

Class that tries to find the best hydrogen label position for 2D depictions. This method decides the position based on the atom and neighbouring atom coordinates. Currently support 4 directions: Right(default), Left, Above, Below
